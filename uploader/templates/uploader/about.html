About

openTeach was created in February 2015. It was a bit of a reaction to this article I wrote. It was mainly a rant about how teachers are eternally locked into products or workflows where we keep re-making resources.

Even though there's lots of amazing resource sites out there, I found that teaching for mainly exam classes, generic resources weren't good enough and I was ironically making all of my own resources from scratch.

If I could have found good quality exam-specific resources that would have been great. That is what this site is for; it aims to be the best curated resources for any given exam board.

Technical

openTeach is written in Python using the Django framework. The source code is available at Github, bug reports and pull requests are welcome. I am good and neither Django or Web design, so any improvements in those areas are very welcome.

If you have a problem and don't know what Github is, contact me on twitter @kieranhogg.
