Lessons

You can combine resources to create a lesson. This will give you both a version to use on the big screen and a version for students to access the resources.



Resources

The site can host different types of resources, files, bookmarks, notes (wiki-style pages) and questions. Everything you might need for a lesson or revision.







Tests

You can assign a set of questions to a lesson task to assess student understanding of the topics you have taught. Students can get feedback instantly.



About

eduResourc.es was created in February 2015 to try to prevent teachers from re-making resources all the time.

Even though there's lots of amazing resource sites out there, I found that teaching for mainly exam classes, generic resources weren't good enough and I was ironically making all of my own resources from scratch.

If I could have found good quality exam-specific resources that would have been great. That is what this site is for; it aims to be the best curated resources for any given exam board.

Technical

eduResourc.es is written in Python using the Django framework. The source code is available at Github, bug reports and pull requests are welcome. I am good and neither Django or Web design, so any improvements in those areas are very welcome.
